Cria uma solução HTML+CSS+JS em que indiques a faixa etária de uma pessoa dada uma idade considerando os seguintes intervalos de idade:
>=0 e <= 2 anos -> bebé;
> 2  e < 12 -> criança;
>= 12 e <= 16 -> adolescente;
> 16 e < 18 -> jovem adulto;
>= 18 e < 50 -> adulto;
>= 50 e <= 65 -> adulto de meia idade;
> 65 -> idoso;
> 100 anos -> lendário;
> 130 anos -> imortal.
